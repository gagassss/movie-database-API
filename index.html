FREE MOVIE STREAMING API

movie-files.herokuapp.com crawls and scraping various websites to find movie information and streaming links which not stored into our database.

GET popular movies

Get a list of the current popular movies, This list updates daily.

Base url: https://movie-files.herokuapp.com/api/movie/popular

Parameter Required Valid options Default value Description
limit *optional (Number 1 - 20) Max 20 queries 20 Limit of queries
page *optional (Number 1 - 1000) Max 1000 1 Specify which page to query, you can use this for pagination

Example: https://movie-files.herokuapp.com/api/movie/popular?page=1&limit=10

GET Top Rated movies

Get the top rated movies.

Base url: https://movie-files.herokuapp.com/api/movie/top-rated

Parameter Required Valid options Default value Description
limit *optional (Number 1 - 20) Max 20 queries 20 Limit of queries
page *optional (Number 1 - 1000) Max 1000 1 Specify which page to query, you can use this for pagination

Example: https://movie-files.herokuapp.com/api/movie/top-rated?page=2&limit=15

GET Search movies

Search for movies.

Base url: https://movie-files.herokuapp.com/api/movie/search

Parameter Required Valid options Default value Description
query *Required String title of searched movie (ex: transformer) Title of searched movies
limit *optional (Number 1 - 20) Max 20 queries 20 Limit of queries
page *optional (Number 1 - 1000) Max 1000 1 Specify which page to query, you can use this for pagination
year *optional Number movie release year

Example: https://movie-files.herokuapp.com/api/movie/search?query=transformer&page=2&limit=15

GET Detail movies

Detail movies.

Base url: https://movie-files.herokuapp.com/api/movie/detail/{id}

Parameter Required Valid options Default value Description
id *Required Valid id you can get from field response {id: integer}

Example: https://movie-files.herokuapp.com/api/movie/detail/550

Important for you to know

Data response table :

Field type Description
id number id of movie content, you will use this to get detail movies information
success boolean explain whether the request was successful or not
dataLength number amount of data movies
data array object list of data movies
playerUrl array links streaming, this is can you use to embed in iframe tag too
